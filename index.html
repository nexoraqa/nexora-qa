How Nexora QA Works
Call Quality Intelligence Platform
📋 The Workflow
1. Submit Evaluation — Assessor opens Score Calculator, selects an agent, scores the call across NC/CC/CB attributes and submits.

2. Auto-Save — Evaluation saves instantly to the app and syncs to Google Sheets in the background. All users see the same data.

3. Dashboard Updates — All charts, scores, and patterns update automatically with every new evaluation.
⚙️ Scoring System
NC (Non-Critical) — 6 attributes worth 30 points total. Each error deducts points.

CC (Critical to Customer) — Any error = automatic fail for that category.

CB (Critical to Business) — Any error = automatic fail for that category.

Overall Score = NC score + 70 points (if no CC or CB errors).
Pass mark = 90%
🎯 Pattern Intelligence
Tracks when the same agent makes the same error across multiple evaluations:

Monitor — CC/CB 1 repeat · Watch this agent
Intervene — CC/CB 2 repeats · Coach now
Escalate — 4+ repeats · Supervisor action required
🔴 At Risk Radar
Shows agents who are struggling right now based on their current scores — not historical patterns.

Below 80% = Critical · 80–89% = At Risk · CC/CB error = flagged regardless of overall score
